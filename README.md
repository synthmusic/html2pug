# HTML2PUG

Installation:

Clone repo

Execute
`npm install`

Then start with
`npm run dev`

After Parcel build is complete there will be a server running at http://localhost:1234

Project for converting html to pug.
Powered by [xhtml2pug](https://github.com/dimensi/html2pug).

-   For develop: `npm run dev`
-   For build: `npm build`
